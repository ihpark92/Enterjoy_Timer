# 아이콘 가이드

이 폴더에 확장프로그램 아이콘을 추가해주세요.

## 필요한 아이콘 크기

- `icon16.png` - 16x16 픽셀 (툴바)
- `icon48.png` - 48x48 픽셀 (확장프로그램 관리 페이지)
- `icon128.png` - 128x128 픽셀 (Chrome 웹 스토어)

## 아이콘 제작 팁

1. PNG 형식 사용 (투명 배경 권장)
2. 심플하고 명확한 디자인
3. 작은 크기에서도 잘 보이도록 디자인
4. 브랜드 컬러 활용
